## 1.2 常用工具类
- LoadingCache（本地缓存）
- Multimap 和 Multiset
- BiMap
- Table（表）
- Sets和Maps（交并差）
- EventBus（事件）
- StopWatch（秒表）
- Files（文件操作）
- RateLimiter（限流器）
- Guava Retry（重试）
